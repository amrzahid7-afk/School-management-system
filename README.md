# 🏫 School Management System (Full-Stack)

An integrated digital solution for managing schools, providing a seamless experience for administrators, teachers, students, and staff. This system covers academic, financial, and administrative aspects in one unified platform.

## 🚀 Project Overview

This project is built using a modern tech stack to ensure high performance and scalability:

- **Backend:** Laravel (RESTful API)
- **Web Dashboard:** React.js (For Administrators)
- **Mobile Application:** Flutter (For Teachers, Students, and Staff)

---

## 🛠 Features & Modules

- **Academic Management:** Manage sessions, classes, and subjects.
- **Attendance System:** Track daily attendance for students and staff.
- **Financial Operations:** Manage tuition fees, salaries, and discounts.
- **Comprehensive Reports:** Generate detailed reports for performance and finances.
- **User Management:** Secure accounts for everyone in the school ecosystem.

---

## 👥 Roles & Permissions

| Platform             | Role           | Access Level                                               |
| :------------------- | :------------- | :--------------------------------------------------------- |
| **Web (React)**      | **Admin**      | Full control over the system, financial data, and reports. |
| **Web (React)**      | **Supervisor** | Academic monitoring and staff management.                  |
| **Mobile (Flutter)** | **Teacher**    | Manage sessions, record attendance, and student grades.    |
| **Mobile (Flutter)** | **Student**    | View schedules, reports, and academic progress.            |
| **Mobile (Flutter)** | **Librarian**  | Manage book inventory and student borrowings.              |

```text
## 📁  Installation & Setup
Backend: Navigate to Project_2_laravel, run
 -composer install ,
 -php artisan migrate ,
 -php artisan db:seed.

Frontend: Navigate to project_2_react, run :
 - npm install
 - npm start.
```
